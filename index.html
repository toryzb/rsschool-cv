Code example:

Create a function combos, that accepts a single positive integer num (30 > num > 0) and returns an array of arrays of positive integers that sum to num. Task link. 

                    
                        def combos(n):
                            if n == 0:
                                return [[]]
                            elif n == 1:
                                return [[1]]
                            else:
                                arr = []
                                arr.append([n])
                                for i in range(1, n):
                                    for k in combos(n-i):
                                        k.append(i)
                                        j = sorted(k)
                                        arr.append(j)
                            res = []
                            for i in arr:
                                if i not in res:
                                    res.append(i)

                            return res
